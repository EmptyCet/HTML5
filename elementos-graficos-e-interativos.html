SVG

SVG (Scalable Vector Graphics) is an XML-based markup language for creating two-dimensional graphics. Different from <canvas> which is an area of pixels manipulated by JavaScript, SVG represents grahic elements such as DOM objects, allowing direct manipulation via CSS or JavaScript.

iFrame

iFrame (inline frame) is an HTML element that allows embedding another HTML page within the current page. Embedding external contents, like youtube videos, or images from image banks. (Not all websites allow embedding)
